Building a One Page Template using bootstrap 4

Starter files was Forked from https://github.com/bradtraversy/bs4starter

##View Demo Here: http://seunzone.com/freeboot4demo/

Includes complete Bootstrap 4.0.0-beta dev environment with gulp and sass

If you are looking for the alpha6 version [click here](https://github.com/bradtraversy/bs4starter_alpha6)

### Version

1.0.0

## Install Dependencies

```bash
npm install 
```

## Compile Sass & Run Dev Server

```bash
npm start
```

Files are compiled into /src

Just Clone or Download, all you need is in the 'src' folder
